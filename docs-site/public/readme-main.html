Installation

    :Mason

Edit lua/config/keymaps.lua:

Adding Plugins

Create a new file in lua/plugins/:

return {
  {
    "plugin/name",
    config = function()
      -- configuration
    end,
  },
}

Modifying Options

Edit lua/config/options.lua to change Neovim settings.
